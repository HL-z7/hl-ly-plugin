# 安装教程
仅Gitee在Yzai根目录执行即可


``` bash
git clone --depth 1 https://gitee.com/fox-glaze/hl-plugin ./plugins/hl-plugin
```

- 一定要配置ffmpeg！不然本插件部分功能会用不了

# 当前功能
