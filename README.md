# 🚀安装教程
仅Gitee在Yzai根目录执行即可


``` bash
git clone --depth 1 https://gitee.com/fox-glaze/hl-ly-plugin ./plugins/hl-ly-plugin
```

# 🐔当前功能
| 功能名称 | 使用方法 | 功能效果 |
|-------| ----- |------ |
| 帮助 | #hl帮助 | 发送本插件帮助图 |
| 天气 | H天气广东 | 天气预报 |
| 多样式戳一戳 | 发送 #HL设置 查看配置 | 多样化 |
| 翻译 | #h翻+要翻译的内容 | 翻译文本 |
| 疯狂星期四 | #随机kfc | 发送KFC文案 |
| 今日运势 | #运势 | 查看今日运势 |
| 表情包/cos | hl列表 | 发送即可查看具体功能 |
| 快捷群管 | #hl帮助 | 查看帮助图查看具体功能 |
| 快捷下载 | 取文件+路径目标文件/取包+路径目标文件夹 | 快捷下载崽的文件/文件夹 |
| 快捷下载（附加） | 开启/关闭私发 | 开启后文件只会发送至主人私信 关闭则发送至触发指令的群聊 |
| 一键群发 | 一键群发+内容（具体功能发送【H群发帮助】查看） | 支持TRSS多账号指定bot号发送 |
| 各种加密 |  |  |
| url加密/解密 | Hurl加密/解密+内容 | 加密/解密 |
| base64加密/解密 | Hbase64加密/解密+内容 | 加密/解密 |
| base64加密/解密 | Hb64加密/解密+内容 | 加密/解密Hex 16进制 不编码 |
| MD5加密 | Hmd5加密+内容 | 加密 |
| Unicode加密/解密 | HUnicode加密/解密+内容 | 加密/解密 |
- 如有功能不能用的情况 请更新插件，什么！？更新插件还是不能用？ 大概率还在想着怎么去修bug，需要点时间，反正等更新就对了
